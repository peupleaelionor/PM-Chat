# PM-Chat Online — Plan du site

> Structure complète du site web PM-Chat Online.

## Arborescence

```
/                          → Page d'accueil (landing)
/device                    → PM-Chat Device (produit matériel)
/setup                     → PM-Chat Setup (déploiement)
/online                    → PM-Chat Online (plateforme numérique)
/technical                 → Vue d'ensemble technique
/documentation             → Accès à la documentation
/documentation/firmware    → Documentation firmware
/documentation/setup       → Guides de déploiement
/documentation/security    → Modèle de sécurité
/roadmap                   → Feuille de route
/contact                   → Contact / Demande pilote
/downloads                 → Téléchargements firmware
```

## Détail des routes

| Route | Titre | Description courte |
|---|---|---|
| `/` | Page d'accueil | Présentation générale, proposition de valeur, accès rapide aux trois piliers |
| `/device` | PM-Chat Device | Présentation du produit matériel, spécifications, cas d'usage |
| `/setup` | PM-Chat Setup | Couche de déploiement, guides de mise en service, provisionnement |
| `/online` | PM-Chat Online | Plateforme numérique compagnon, documentation, téléchargements |
| `/technical` | Vue d'ensemble technique | Architecture, protocole, sécurité, topologie maillée |
| `/documentation` | Documentation | Hub d'accès à toute la documentation technique |
| `/documentation/firmware` | Documentation firmware | Référence firmware, API, structure du code |
| `/documentation/setup` | Guides de déploiement | Flashage, provisionnement, appairage, maintenance |
| `/documentation/security` | Modèle de sécurité | Chiffrement, gestion des clés, modèle de menace |
| `/roadmap` | Feuille de route | État actuel, priorités à court terme, vision long terme |
| `/contact` | Contact | Formulaire de demande pilote et contact général |
| `/downloads` | Téléchargements | Binaires firmware, code source, historique des versions |

## Notes

- Toutes les pages partagent un en-tête commun avec navigation vers les trois piliers (Device, Setup, Online).
- Le pied de page inclut les liens vers la documentation, le contact et les téléchargements.
- La navigation est conçue pour être sobre et directe, sans menus profonds.
